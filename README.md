This project is a heart shaped PCB featuring multiple LEDs arranged around the edges. It is designed as a biginner-friendly elecronics project to learn PCB design, circuit connections, and components placement.

13 LEDs placed around the boarder
Push-button switch for control
Compact layout
Designed using KiCad

HOW IT WORKS?
The circuit connects multiple LEDs with resisitors and a power source. When powered, the LEDs blinks.
A push button is included to control the circuit.

HOW TO USE?
1.Manufacture the PCB using Gerber files
2.Solder all components (LEDs, resistors, switch, battery holder.)

FILES INCLUDED 
1.Schematic design
PCB layout
Gerber files for manufacturing
